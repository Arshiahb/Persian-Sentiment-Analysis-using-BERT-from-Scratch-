# Persian Sentiment Analysis with BERT

این پروژه یک سیستم تحلیل احساسات (Sentiment Analysis) پیشرفته برای متون فارسی است که به صورت ۳‌کلاسه (مثبت، منفی، خنثی) طراحی شده است. تمرکز این پروژه بر روی پایداری مدل، استفاده از مکانیزم‌های آستانه‌گذاری (Thresholding) و معماری ماژولار جهت استنتاج سریع است.

## ویژگی‌های فنی
- **معماری:** مبتنی بر BERT با سفارشی‌سازی برای زبان فارسی.
- **بهینه‌سازی:** استفاده از `Focal Loss` برای مدیریت عدم توازن کلاس‌ها و جلوگیری از Overfitting روی کلاس خنثی.
- **سیستم Gating:** پیاده‌سازی مکانیزم آستانه‌گذاری پویا برای کلاس خنثی (Neutral) جهت افزایش Precision.
- **طراحی ماژولار:** تفکیک کاملِ Tokenizer، مدل، و اسکریپت‌های آموزش از محیط استنتاج (Inference).
- **پایداری:** ذخیره‌سازی هوشمند Vocab و Config جهت حذف وابستگی به داده‌های آموزشی در زمان Inference.
